# Ignite Landing Page

A modern, black-themed landing page for the Ignite mobile app - a self-improvement app that gamifies discipline through custom 70-day challenges.

## Features

- **Modern Black Theme**: Sleek dark design with orange gradient accents
- **Responsive Design**: Fully responsive across all device sizes
- **Interactive Animations**: Smooth scroll animations and parallax effects
- **Phone Mockup**: Visual representation of the app interface
- **Sections Included**:
  - Hero section with key messaging
  - Features showcase
  - How it works (3-step process)
  - Download section with app store buttons
  - Footer with navigation links

## Getting Started

1. Open `index.html` in your web browser
2. No build process required - it's a static HTML page

## Customization

### Colors
Edit the CSS variables in `styles.css`:
- `--bg-primary`: Main background color
- `--accent-primary`: Primary accent color
- `--accent-gradient`: Gradient colors

### Content
- Edit `index.html` to update text content
- Replace placeholder download links with actual app store URLs
- Update footer links as needed
